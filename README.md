# Ticketsystem

Ein einfaches webbasiertes Ticketsystem mit PHP und MariaDB, das es Benutzern ermöglicht, Tickets zu erstellen, zu verfolgen und zu verwalten.

## Installation

### Voraussetzungen

- **Webserver**: Nginx
- **PHP**: Version 8.3 oder höher
- **MariaDB**: Datenbank
- **Git**: Version 2.0 oder höher
